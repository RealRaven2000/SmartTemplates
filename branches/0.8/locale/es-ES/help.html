Variables fijas
%ownname% = Su nombre
%ownmail% = Su dirección de correo
%date% = Fecha y hora (las mismas que el formato original de la cabecera del correo)
%to(mail)% = Dirección de correo del destinatario
%from(mail)% = Dirección de correo electrónico del remitente
%cc(mail)% = Dirección de correo del destinatario
%sig% = Firma (de ajustes de la cuenta de la TB)
%sig(2)% = Firma (de ajustes de la cuenta de la TB pero sin el ' --')

Variables de destinatario (Para)

(Cuando no se disponga del nombre, se mostrará la dirección de correo electrónico)

%to% = Nombre del destinatario (predeterminado)
%to(firstname)% = Nombre de pila del destinatario
%to(name)% = Apellido del destinatario

Variables de emisor (De)

(Cuando no se disponga del nombre, se mostrará la dirección de correo electrónico)

%from% = Nombre de remitente (predeterminado)
%from(firstname)% = Nombre de pila del remitente
%from(name)% = Apellido del remitente

Variables de destinatario en copia (CC)

(Cuando no se disponga del nombre, se mostrará la dirección de correo electrónico)

%cc% = Nombre de destinatario (predeterminado)
%cc(firstname)% = Nombre de pila del destinatario
%cc(name)% = Apellido del destinatario

Formato especial
Para: %to(name)%{
Cc: %cc(name)%}

¡Sólo se mostrará la frase"{Cc: %cc(name)%}"

si se puede sustituir la variable %cc(name)!



Asunto
%subject% = Asunto (iguales tienen gusto de Mailheader original)
%subject(2)% = Asunto (iguales tienen gusto de Mailheader original pero sin ' --')

Variables espciales
%Message-ID% = ID de mensaje
%Newsgroups% = Grupos de noticias
%X-Mailer% = X-Mailer

Variables fijas para formatos de fecha y hora y zona horaria

(Se pueden combinar con %X:=sent%)

%datelocal% = Fecha y hora (formato local)
%dateshort% = Fecha y hora abreviadas (p.ej. 01.01.2000 1:23:45)
%date_tz% = Zona horaria ("+0100";"+0200"; ...)
%tz_name% = GMT; PST; EST; ...

Variables para formatos personalizados de fecha y hora

(Se pueden combinar con %X:=sent%)

%A% / %a% = Día de la semana ("Viernes" / "Vie.")
%d% / %e% = Día del mes ("01 a "31" / "1" a "31")
%m% / %n% = Mes ("01" a "12" / "1 a 12")
%B% / %b% = Mes ("Enero" / "Ene.")
%Y% = Año ("1970")
%M% = Minutos ("00" a "59")
%S% = Segundos ("00" a "59")
%H% / %k% = Hora en formato de 24 horas ("00" a "23" / "0" a "23")
%I% / %l% = Hora en formato de 12 horas ("01" a "12" / "1" a "12")
%p% = "AM" o "PM"
%p(1)% = (a.m.)
%p(2)% = (A.M.)
%p(3)% = (AM)

Uso de %X:=sent%
Las variables descritas anteriormente normalmente adoptarán los valores actuales.
En cambio, cuando se combinan con %X:=sent%, adoptarán los valores previos.

Ejemplo de una cadena de texto larga: %X:=sent%%d%.%m%.%Y% a las %H%:%M%: Resultado: 16.02.2011 a las 12:15:

Además, puede usar las siguientes variables: %datelocal% = Fecha y hora (formato local) %dateshort% = Fecha y hora abreviadas (p.ej. 01.01.2000 1:23:45) %date_tz% = Zona horaria ("+0100";"+0200"; ...) para crear una cadena de texto corta.

Ejemplo de cadena de texto corta:
%X:=sent%%datelocal%
Resultado: Miércoles, 16.02.2011 12:15:25
or
%X:=sent%%dateshort%
Resultado: 16.02.2011 12:15:25
%date_tz% = Zona horaria ("+0100";"+0200"; ...)
En los dos casos puede usar además la variable de zona horaria.

Notas sobre las plantillas de HTML
¡Por favor, utilice únicamente caracteres UTF-8 (Unicode) en las plantillas de HTML!

- Caracteres especiales " , & , < , >
Unicode: &quot; , &amp; , &lt; , &gt;

- Diéresis Ä , ä , Ü , ü , Ö , ö , ß
Unicode: &Auml; , &auml; , &Uuml; , &uuml; , &Ouml; , &ouml; , &szlig;

- Salto de línea
Tiene que usar <br> para el salto de línea
cuando no utilice 'Sustituir salto de línea por <br>'.
