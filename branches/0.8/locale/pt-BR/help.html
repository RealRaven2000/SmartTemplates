Variáveis Básicas
%ownname% = Seu nome
%ownmail% = Seu endereço de e-mail
%date% = Data e hora (igual ao encontrado no cabeçalho da mensagem)
%to(mail)% = Nome e endereço do destinatário
%from(mail)% = Nome e endereço do remetente
%cc(mail)% = Nome e endereço do destinatário
%sig% = Assinatura (da TB Conta Settings)
%sig(2)% = Assinatura (da TB Conta Settings mas sem o '--')

Variáveis de Destinatário (Para)

(Quando o nome não estiver disponível, o endereço de e-mail será exibido no lugar)

%to% = Destinatário (padrão)
%to(firstname)% = Primeiro nome do destinatário
%to(name)% = Nome do destinatário

Variáveis de Remetente (De)

(Quando o nome não estiver disponível, o endereço de e-mail será exibido no lugar)

%from% = Remetente (padrão)
%from(firstname)% = Primeiro nome do remetente
%from(name)% = Nome do remetente

Variáveis de Destinatário (Cc)

(Quando o nome não estiver disponível, o endereço de e-mail será exibido no lugar)

%cc% = Destinatário (padrão)
%cc(firstname)% = Primeiro nome do destinatário
%cc(name)% = Nome do destinatário

Formato Especial
Para: %to(name)%{
Cc: %cc(name)%}

A frase "{Cc: %cc(name)%}" somente será exibida, se a variável %cc(name) não for vazia!

Assunto
%subject% = Assunto (mesmo como mailheader originais)
%subject(2)% = Assunto (mesmo como editor Mensagem TB)

Variáveis Especiais
%Message-ID% = Identificador da mensagem
%Newsgroups% = Newsgroups
%X-Mailer% = X-Mailer

Variáveis para Data e Hora

(podem ser combinadas com %X:=sent%)

%datelocal% = Data e hora (formato local)
%dateshort% = Data e hora curta (por exemplo, 01/01/2000 1:23:45)
%date_tz% = Fuso horário ("+0100";"+0200"; ...)
%tz_name% = GMT; PST; EST; ...

Variáveis para Data e Hora Personalizáveis

(podem ser combinadas com %X:=sent%)

%A% / %a% = Dia da semana ("Domingo" / "Dom.")
%d% / %e% = Dia do mês ("01 to "31" / "1" to "31")
%m% / %n% = Mês (de "01" a "12" / de "1" a "12")
%B% / %b% = Nome do mês ("Janeiro" / "Jan.")
%Y% = Ano ("1970")
%M% = Minutos (de "00" a "59")
%S% = Segundos (de "00" a "59")
%H% / %k% = Hora, no formato de 24 horas (de "00" a "23" / de "0" a "23")
%I% / %l% = Hora, no formato de 12 horas (de "01" a "12" / de "1" a "12")
%p% = "AM" or "PM"
%p(1)% = (a.m.)
%p(2)% = (A.M.)
%p(3)% = (AM)

Uso do %X:=sent%
As variáveis acima são exibidas normalmente com os valores atuais.
Quando combinadas com %X:=sent%, elas são exibidas com os valores anteriores.

Exemplo com variáveis personalizáveis:
%X:=sent%%d%/%m%/%Y% às %H%:%M%
Resultado: 16/02/2011 às 12:15

Exemplos com variáveis fixas:
%X:=sent%%datelocal%
Resultado: Quarta-feira, 16/02/2011 12:15:25
ou
%X:=sent%%dateshort%
Resultado: 16/02/2011 12:15:25

Observações para Modelos em HTML
Por favor, use somente caracteres UTF-8 (Unicode) nos modelos em HTML!


- Caracteres especiais:
" : &quot;, & : &amp;, < : &lt;, > : &gt;
- Letras com acentos, til, cedilha:
Á : &Aacute;, É : &Eacute;, Í : &Iacute;, Ó : &Oacute;, Ú : &Uacute;, á : &aacute;, é : &eacute;, í : &iacute;, ó : &oacute;, ú : &uacute;, Â : &Acirc;, Ê : &Ecirc;, Ô : &Ocirc;, â : &acirc;, ê : &ecirc;. ô : &ocirc;, Ã : &Atilde;, Õ : &Otilde;, ã : &atilde;, õ : &otilde;, À : &Agrave;, à : &agrave;, Ç : &Ccedil;, ç : &ccedil;
- Quebra de linha:
Você precisa usar <br> para inserir uma quebra de linha quando a opção 'Substituir quebra de linha por <br>' não estiver selecionada.
