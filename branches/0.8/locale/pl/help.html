Zmienne podstawowe
%ownname% = Twoje imię
%ownmail% = Twój adres e-mail
%date% = Data i czas (taki sam, jak oryginał nagłówka)
%to(mail)% = Adres e-mail odbiorcy
%from(mail)% = Adres e-mail nadawcy
%cc(mail)% = Adres e-mail odbiorcy
%sig% = Podpis (z uwzględnieniem TB ustawienia)
%sig(2)% = Podpis (z uwzględnieniem TB ustawienia ale bez '--')

Zmienne odbiorcy (Do:)

(Gdy nazwisko jest niedostępne, będzie wyświetlany adres e-mail)

%to% = Nazwa odbiorcy (domyślna)
%to(firstname)% = Imię odbiorcy
%to(name)% = Nazwisko odbiorcy

Zmienne nadawcy (Nadawca:)

(Gdy nazwisko jest niedostępne, będzie wyświetlany adres e-mail)

%from% = Nazwa nadawcy (domyślna)
%from(firstname)% = Imię nadawcy
%from(name)% = Nazwisko nadawcy

Zmienne odbiorcy kopii (Kopia:)

(Gdy nazwisko jest niedostępne, będzie wyświetlany adres e-mail)

%cc% = Nazwa odbiorcy (domyślna)
%cc(firstname)% = Imię odbiorcy
%cc(name)% = Nazwisko odbiorcy

Format specjalny
Do: %to(name)%{
Kopia: %cc(name)%}

Fraza "{Cc: %cc(name)%}" będzie wyświetlana tylko, jeśli
zmienna %cc(name) może być zamieniana!

Temat
%subject% = Temat (sam jak oryginalny nagłówek listu)
%subject(2)% = Temat (sam jako edytor wiadomości TB)

Zmienne specjalne
%Message-ID% = ID wiadomości
%Newsgroups% = Grupy dyskusyjne
%X-Mailer% = X-Mailer

Zmienne dla formatu daty i strefy czasowej

(Mogą być łączone z %X:=sent%)

%datelocal% = Data i czas (lokalny format)
%dateshort% = Data i czas (krótki format)) (np. 01.01.2000 1:23:45)
%date_tz% = Strefa czasowa ("+0100";"+0200"; ...)
%tz_name% = GMT; PST; EST; ...

Zmienne dla dostosowania formatu daty

(Mogą być łączone z %X:=sent%)

%A% / %a% = Dzień tygodnia („Piątek” / „Pi”)
%d% / %e% = Dzień miesiąca („01” do „31” / „1” do „31”)
%m% / %n% = Miesiąc („01” to „12” / „1 do 12”)
%B% / %b% = Miesiąc („Styczeń” / „Sty”)
%Y% = Rok („1970”)
%M% = Minuty („00” to „59”)
%S% = Sekundy („00” to „59”)
%H% / %k% = godzina, format 24-godzinny („00” do „23” / „0” do „23”)
%I% / %l% = godzina, format 12-godzinny („01” do „12” / „1” do „12”)
%p% = „AM” lub „PM”
%p(1)% = (a.m.)
%p(2)% = (A.M.)
%p(3)% = (AM)

Użycie %X:=sent%
Zmienne określane przez aktualne wartości będą zmiennymi górnego poziomu.
Jeśli będą łączone z %X:=sent%, będą zmiennymi dolnego poziomu.

Przykład długiego formatu:
%X:=sent%%d%.%m%.%Y% o godz. %H%:%M%:
Wynik: 16.02.2011 o godz. 12:15:


Ponadto można używać następujących zmiennych:
%datelocal% = Data i czas (lokalny format)
%dateshort% = Data i czas (krótki format)) (np. 01.01.2000 1:23:45)
%date_tz% = Strefa czasowa ("+0100";"+0200"; ...)
do określenia krótkiego formatu.

Przykład krótkiego formatu:
%X:=sent%%datelocal%
Wynik: Środa, 16.02.2011 12:15:25
or
%X:=sent%%dateshort%
Wynik: 16.02.2011 12:15:25
%date_tz% = Strefa czasowa ("+0100";"+0200"; ...)
W obu wariantach można dodatkowo użyć zmiennej strefy czasowej.

Informacje dla stosowania szablonów HTML
W szablonach HTML, należy używać kodowania znaków UTF-8 (Unicode)!

- Znaki specjalne: " , & , < , >
Unicode: &quot; , &amp; , &lt; , &gt;

- Polskie ogonki: np. ą , ę , ł , ż
Unicode: &#261; , &#281; , &#322; , &#380;

- Nowe wiersze:
Należy używać <br> dla nowych wierszy, gdy nie jest używana funkcja
'Zamieniaj nowy wiersz znakiem <br>'.
