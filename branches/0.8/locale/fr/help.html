Variables fixes
%ownname% = Votre nom
%ownmail% = Votre adresse mail
%date% = Date et heure (ceux du message original)
%to(mail)% = Adresse email du destinataire
%from(mail)% = Adresse email de l'envoyeur
%cc(mail)% = Adresse email du destinataire en copie
%sig% = Signature (de la TB paramètres de compte)
%sig(2)% = Signature (de la TB paramètres de compte mais sans '--')

Les destinataires (Pour)

(Quand le nom n'est pas disponible, cette adresse sera affichée)

%to% = Destinataire (Défault)
%to(firstname)% = Prénom du destinataire
%to(name)% = Nom du destinataire

L'envoyeur (From)

(Quand le nom n'est pas disponible, cette adresse sera affichée)

%from% = Envoyeur (Défault)
%from(firstname)% = Prénom de l'envoyeur
%from(name)% = Nom de l'envoyeur

Destinataires en copie (CC)

(Quand le nom n'est pas disponible, cette adresse sera affichée)

%cc% = Destinataires en copie (Default)
%cc(firstname)% = Prénom du destinataire en copie
%cc(name)% = Nom du destinataire en copie

Formats spéciaux
To: %to(name)%{
Cc: %cc(name)%}

La phrase "{Cc: %cc(name)%}" s'affichera seulement
si la variable %cc(name) peut être remplacée !

Sujet
%subject% = Sujet (mêmes aiment Mailheader original)
%subject(2)% = Sujet (mêmes que le rédacteur de message de TB)

Variables Spéciales
%Message-ID% = ID du message
%Newsgroups% = Newsgroups
%X-Mailer% = X-Mailer

Variables fixes pour le format de date et la zone horaire

(Peut être combinée avec %X:=sent%)

%datelocal% = Date et heure (format local)
%dateshort% = Date et heure courtes (ex : 01.01.2000 1:23:45)
%date_tz% = Zone horaire ("+0100";"+0200"; ...)
%tz_name% = GMT; PST; EST; ...

Les variables pour personnaliser le format de date

(Peut être combiné avec %X:=sent%)

%A% / %a% = Jour de la semaine ("Vendredi" / "vend.")
%d% / %e% = Jour du mois ("01 à "31" / "1" à "31")
%m% / %n% = Numéro du mois ("01" à "12" / "1 to 12")
%B% / %b% = Mois ("January" / "Jan.")
%Y% = Année ("1970")
%M% = Minutes ("00" à "59")
%S% = Secondes ("00" à "59")
%H% / %k% = Heure, Format 24h ("00" à "23" / "0" à "23")
%I% / %l% = Heure, Format 12h ("01" à "12" / "1" à "12")
%p% = "AM" ou "PM"
%p(1)% = (a.m.)
%p(2)% = (A.M.)
%p(3)% = (AM)

Utilisation de %X:=sent%
Les variables ci-dessus affichent normalement les valeurs courantes.
Lorsque vous les combinez avec %X:=sent%, les valeurs seront alors dans le passé.

Exemple d'un format long:
%X:=sent%%d%.%m%.%Y% à %H%:%M%
Résultat : 16.02.2011 à 12:15

En plus, vous pouvez utiliser ces variables:
%datelocal% = Date et heure (format local)
%dateshort% = Date et heure courtes (ex : 01.01.2000 1:23:45)
%date_tz% = Zone horaire ("+0100";"+0200"; ...)

pour configurer un format court.

Exemple de format court:
%X:=sent%%datelocal%
Résultat : Mercredi, 16.02.2011 12:15:25
ou
%X:=sent%%dateshort%
Résultat : 16.02.2011 12:15:25
%date_tz% = Timezone ("+0100";"+0200"; ...)
Dans les deux variantes, vous pouvez ajouter la variable indiquant la zone horaire.

Notes pour l'utilisation de HTML
Merci de n'utiliser que le jeu de caractère UTF-8 (Unicode) dans le HTML !

- caractères spéciaux: " , & , < , >
Unicode: &quot; , &amp; , &lt; , &gt;

- Trémas: Ä , ä , Ü , ü , Ö , ö , ß
Unicode: &Auml; , &auml; , &Uuml; , &uuml; , &Ouml; , &ouml; , &szlig;

- Nouvelle ligne:
Vous devez utiliser <br> pour les nouvelles lignes quand
'Remplacez les sauts de ligne avec <br>' n'est pas coché.
