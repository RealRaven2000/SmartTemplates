SmartTemplates :: Frequently Asked Questions

  • Can I use an Alias Email for my License Key?

    No. The license validation mechanism does not work with Alias addresses - only Default Identities are supported. The reason is that the license key (and the associated email address) are not transmitted / stored / processed anywhere outside of the Thunderbird Application - as this is the safest and most private way with 100% uptime which is also guaranteed to work if you are always offline. The associated email does not need to be able to receive or send email either. The Add-on and my websites are fully GDPR compliant, and no data at all is shared with 3rd parties (outside of what the web shop at fastspring needs to secure and process your payment information).

  • Can i use SmartTemplates for a auto responder?

    No, for technical reasons this is not possible.
    All contents which we modify will be created only if you open the e-Mail inside the compose window and this can't be automated.

  • Can i create templates or signatures per recipient?

    No this is not possible.
    SmartTemplates bind Templates and signatures at the Account ID only which you have set up in your Account Settings.

    See our examples page

  • I don't have any space to type my reply or forwarded message?

    You just need to add some line-breaks <br><br> to the top of your template!

    See our examples page

  • I am using %sig% in my template but it isn't showing up?

    'Include Signature on Reply (or Forward)' must be checked in Thunderbird options for the %sig% variable to work as expected.

  • Why does SmartTemplates only work in Inbox with IMAP Accounts?

    Go to Tools ⇒ Account Settings ⇒ Synchronization & Storage.
    Make sure 'Keep messages for this account on this computer' is checked, then click 'Advanced...'. Make sure all folders are checked.

  • Why does my language not work properly in Prereleases & Prototypes?

    All Prereleases & Prototypes include the full en-US and de locale for the current pre release.
    Other locales can cause trouble as the translation is usually still ongoing! In the heat of the moment we often post prerelease versions where strings are missing in other languages.

  • Why are some texts of SmartTemplates in English although I have a different language?

    Our translation team at Babelzilla is notified before a new version gets released,
    but if a translator can not make the deadline, we put in the English Phrases so that we can publish our new version on ATN.
    Depending on the translator, new strings will usually be translated in the following version.

    See our examples page

  • Why are attached images not displayed on replies or forwarded messages?

    This behaviour is by design, if you have 'Message Synchronizing' in Thunderbird disabled.
    The images in this case are not local on the computer and must be downloaded each time he view/reply/forward the message.
    When you click reply, Thunderbird starts downloading the message and it can take longer depending on how big the email or attachment is.
    To solve this Problem: Go to Tools ⇒ Account Settings ⇒ Synchronization & Storage and turn them ON to work as expected.

    See our examples page

  • Why can't I import your example templates from the examples page?

    Import and Export (Template Sharing) is a new feature which we have planned for SmartTemplates 1.0.
    If you use an older Version of SmartTemplates you can copy & paste our example templates...

    See our examples page

  • Is it possible to use JavaScript within the template?

    While it isn't officially supported this was implemented way back in 2014 by Benito van der Zander in [bug 25676]. This will execute some (restricted) JavaScript function to generate and insert a string in place.
    Here is an example: 

    %{% 
  ( () => {
  var date = new Date();
  if (date.getHours() < 12) {return "Good morning";}
  else { return "Good afternoon"; }
  } )() 
%}%
