  • A simple New message template


    TIP: This template is intended for writing new messages but is also valid for reply and forward. When in a new email, certains fields (such as from, identity and subject) cannot usually be determined directly after opening composer. Instead they will show as interactive fields that can be updated with a single click. To completely remove the interactive field either send of the mail or click the button

    <p>Dear %recipient(name)%,</p>
    <p>%cursor%</p>
    <p>Yours sincerely, <br>
    &nbsp;&nbsp;%identity(name)%</p>
    %sig%

Encodings

To be able to include a wide selection of different characters in the signature, the routine that reads it in will assume that it is stored as UTF-8. Please make sure to save your signature files in this format, especially when using special (unicode) characters. Alternatively, you can force a different encoding by changing the configuration setting:

  • extensions.smartTemplate4.signature.encoding
